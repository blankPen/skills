# skills

Claude Code 技能集，提升日常工作效率。

## 前置要求

- 已安装 Node.js 环境
- 能够运行 `npx bun` 命令

## 安装

### 快速安装（推荐）

```bash
npx skills add blankPen/skills
```

## 可用技能

### cursor-insights

生成分析 Cursor Agent 会话的报告，用户想要生成一份会话的分析报告时调用
会扫描本地 Cursor 的会话记录，并且分析产出一份 `report.html` 分析报告，给出优化和改进建议；

```bash
/cursor-insights 
```
