# 黄金实时价格查询

## API 信息
- Base URL: http://49.235.61.172:3003
- 推荐接口: /api/summary（综合摘要，信息最全）

## Steps

1. **调用 API 获取数据**
   - 使用 web_fetch 调用 `http://49.235.61.172:3003/api/summary`
   - 或使用 /api/ai/snapshot 获取自然语言摘要

2. **解析返回数据**
   - 提取当前价格 (price)
   - 提取 24 小时涨跌 (change24h, changePercent24h)
   - 提取 24 小时高低均价 (high24h, low24h, average24h)
   - 提取时间戳 (timestamp)

3. **格式化输出**
   - 标题：黄金实时价格 (XAUUSD)
   - 当前价格（大字突出）
   - 24 小时涨跌（涨用绿色，跌用红色）
   - 24 小时高低均价
   - 更新时间

4. **可选增强**
   - 根据涨跌给出简单建议（如"上涨趋势""下跌趋势"）
   - 告知数据更新时间

## Output Example
```
🥇 黄金实时价格 (XAUUSD)

💰 当前价格: $2,034.50
📈 24小时涨跌: +$12.30 (+0.61%)

📊 24小时统计:
   最高: $2,038.20
   最低: $2,020.00
   均价: $2,028.60

🕐 更新时间: 2026-03-04 15:40:00
```
