# 风格指南与常用提示词

## 常用风格关键词

| 风格 | 关键词 | 适用场景 |
|-----|-------|---------|
| 写实 | realistic, photorealistic, 8K, detailed | 人物、产品、风景 |
| 插画 | illustration, digital art, vector style | 头像、图标、海报 |
| 水彩 | watercolor, soft edges, artistic | 艺术创作、封面 |
| 扁平 | flat design, minimalist, clean | UI设计、图标 |
| 赛博朋克 | cyberpunk, neon, futuristic | 科技感设计 |
| 动漫 | anime style, manga, cel shading | 二次元风格 |
| 油画 | oil painting, impressionist, brushstrokes | 艺术创作 |

## 光线关键词

| 光线 | 关键词 | 效果 |
|-----|-------|------|
| 柔和 | soft lighting, diffused light | 柔和氛围 |
| 逆光 | backlight, rim light, silhouette | 轮廓感 |
| 影棚 | studio lighting, professional lighting | 商业质感 |
| 黄金时段 | golden hour, warm sunlight | 温暖氛围 |
| 戏剧性 | dramatic lighting, chiaroscuro | 强烈对比 |

## 构图关键词

| 构图 | 关键词 | 效果 |
|-----|-------|------|
| 特写 | close-up, portrait, headshot | 突出主体 |
| 全景 | wide shot, landscape, panoramic | 展现全貌 |
| 俯视 | top-down view, bird's eye view | 平铺展示 |
| 低角度 | low angle, worm's eye view | 震撼感 |

## 质量增强词

建议在所有提示词末尾添加：
```
high quality, detailed, 4K, professional
```

## 负面提示词（避免生成）

如 API 支持，可使用 negative_prompt：
```
blurry, low quality, distorted, watermark, text errors
```
